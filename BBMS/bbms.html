[ SYSTEM_OVERVIEW ]

The Blood Bank Management System (BBMS) is a full-stack project that simplifies the process of managing blood donors, requests, and stock availability. It ensures transparency, quick access to donor data, and efficient blood request handling.

I designed and implemented the backend logic for donor registration, request handling, and stock updates. Additionally, I built the frontend UI and integrated it with the database for real-time updates.

The main challenge was maintaining consistency between stock levels and requests. Handling concurrent requests required transaction management in MySQL. This project strengthened my knowledge in database design, secure PHP coding, and responsive UI practices.
